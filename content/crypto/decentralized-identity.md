# 去中心化身份 DID 有什么用？

去中心化身份希望让用户用钱包、凭证或链上记录证明身份、资格或声誉，而不是完全依赖中心化账号系统。

可能用途：

- 证明某项资格或参与记录；
- 降低重复注册和女巫攻击；
- 在不同应用之间携带声誉；
- 用零知识证明减少隐私暴露。

风险也很明显：

1. 身份与钱包绑定后，隐私可能下降。
2. 凭证发行方仍可能中心化。
3. 错误或污点记录可能难以清除。
4. 项目可能把 DID 包装成代币叙事。

新手可以把 DID 理解为“链上身份和声誉工具”，但不要因为概念宏大就忽略真实采用、隐私边界和商业模式。
