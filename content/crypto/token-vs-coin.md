# Coin 和 Token 有什么区别？

Coin 通常是某条区块链的原生资产，例如 BTC 和 ETH；Token 则通常由智能合约在已有网络上发行。

完整词条：[Coin 和 Token 有什么区别？](https://xiaoyinsi.com/wiki/crypto/token-vs-coin)

原生 Coin 常用于支付网络手续费和维护共识，Token 则可代表治理权、稳定币、权益或其他应用资产。

这个分类不说明投资质量。拥有自己的主链不代表更安全，在成熟网络发行 Token 也不代表没有价值。

转账前应确认资产所在网络、合约地址和手续费资产，避免把同名 Token 发送到不兼容的链。
