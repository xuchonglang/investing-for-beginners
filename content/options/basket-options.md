# Basket Option 是什么？

Basket Option 的标的是一篮子资产，而不是单只股票或指数。收益取决于篮子整体表现，有时还会受权重、相关性和表现最差资产影响。

分析重点：

1. 篮子包含哪些资产。
2. 每个资产权重是多少。
3. 收益看平均表现、最佳表现还是最差表现。
4. 是否存在汇率或行业集中风险。
5. 能否在二级市场退出。

相关性很重要。如果篮子资产高度相关，分散效果有限；如果结构按最差资产结算，一个拖后腿标的就可能决定整体收益。

新手遇到篮子型结构化产品，不要只看“多资产分散”。真正要看的是收益公式和最坏情景。
