# Box Spread 为什么不是新手套利？

Box Spread 通过同时构造牛市价差和熊市价差，理论上形成接近固定现金流的结构。它常被用来理解无套利定价，也可能用于融资。

问题在于，现实交易不是课本：

- 多腿成交可能产生滑点；
- 美式期权存在提前行权风险；
- 保证金和券商规则可能变化；
- 利率、费用和税务会影响结果；
- 流动性差时很难按理论价平仓。

新手不要看到“无风险套利”就尝试。很多 Box Spread 的事故不是公式错了，而是交易规则、行权机制和保证金约束被忽略。

如果不能逐腿解释现金流和最坏情况，就不要下单。
