# Call Ladder 是什么？

Call Ladder 通常是买入一个较低执行价 Call，同时卖出两个更高执行价 Call，形成有限成本或净收入的看涨但上方有风险结构。

它适合温和上涨观点，不适合无限看涨。

交易前一定要看：

1. 最大收益区间在哪里。
2. 超过上方执行价后风险如何扩大。
3. 是否存在裸卖 Call 风险。
4. 保证金要求是否可承受。
5. 到期前是否有跳空事件。

很多人被低成本吸引，却忽略标的如果大涨，卖出的额外 Call 会造成巨大亏损。

新手如果要表达看涨，通常更适合先理解普通买 Call、牛市价差，再考虑 Ladder 这类不对称结构。
