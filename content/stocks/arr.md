# ARR 年化经常性收入怎么看？

ARR 常用于 SaaS 公司，表示把当前订阅收入年化后的经常性收入规模。它能帮助投资者观察订阅业务的稳定收入基础。

看 ARR 时注意：

- ARR 增长不等于现金马上到账；
- 新客户、续费、扩容和降级要分开看；
- 高 ARR 也可能伴随高获客成本；
- 净收入留存下降时，ARR 增速可能会继续放缓。

实操问题：

1. ARR 增速是否持续下降？
2. 客户留存和净收入留存是否稳定？
3. 销售费用换来的 ARR 是否划算？
4. 剩余履约义务和递延收入是否支持增长？

ARR 是分析订阅公司很有用的入口，但不能替代现金流、利润率和获客效率。
